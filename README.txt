
Install ReactJS

On the terminal: npm install -g create-react-app

Follow the steps on the terminal to start the servers:

Back-end server
	1. cd back-end
	2. npm install
	3. npm start

Front-end server
	1. cd front-end
	2. npm install
	3. npm start

Kafka Back-end server
        1. cd Kafka-back-end
        2. npm install
        3. npm start
